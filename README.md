# atm-mobileapp

## Installation

- Clone the repository
- Install react-native-cli & set android or IOS environments
- > npm install
- > ./node_modules/.bin/rn-nodeify --hack --install

## Run Application
- Connect a device in developer mode or use an emulator
- > react-native start
- In other terminal: 
  > react-native run-android
